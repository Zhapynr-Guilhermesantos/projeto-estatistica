# projeto-estatistica
Projeto de modelagem estatística

Descrição:
Esse projeto é uma análise exploratória e de investigação de relacionamento entre os dados, o foco do dataset utilizado é de saude mental em relação com as redes sociais e tempo de tela 

Instruções:
utilize o Google Colab 3.11 da build de 2025.07, para não ter conflito com a biblioteca pycaret

no Google Colab siga os passos antes de rodar o código:
ambiente de execução -> alterar o tipo de ambiente de execução -> Versão do ambiente de execução -> selecionar versão 2025.07

fonte: https://www.kaggle.com/datasets/prince7489/social-media-and-mental-health-balance
